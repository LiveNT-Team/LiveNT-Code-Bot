# Reference

Этот файл содержит полное техническое задание

## Список административных команд

### `settings`

Выводит настройки сервера

### `set activist_messages_count <количество>`

Устанавливает количество сообщений, необходимых для получения роли активист

### `set activist_role <роль>`

Устанавливает роль активист

### `disable extraditing_activist`

Выключает выдачу роли активист

### `enable extraditing_activist`

Включает выдачу роли активист

### `set greetings channel <канал>`

Устанавливает канал для приветствия новых пользователей

### `disable greetings`

Выключает приветствия для новых пользователей

### `enable greetings`

Включает приветствия для новых пользователей

### `set ai channel [<канал>]`

Устанавливает канал как канал для общения с ии (если параметр канал не указан будет использоваться текущий канал)

### `enable ai`

Включает функции общения с ии

### `disable ai`

Выключает функции общения с ии

## Список общих команд

### `help`

Вывод справочник по командам. Можно реализовать с использование разделов и Select меню.

### `change personality <личность>`

Меняет личность бота

> [!NOTE]
> Эта команда устанавливает личность бота только для текущего участника. То есть бот будет общаться с участником используя личность, которую он установил. Это правило распространяется так же на ai канал.

### `get personalities`

Выводит список доступных личностей бота

### `stats [<участник>]`

Выводит количество сообщений которые пользователь отправил на сервере

### @упоминание_бота <текст запроса>

Отправляет запрос к ИИ

### Выдача роли активист

Выдача роли активист при отправке более чем 199 сообщений


### Система банов и мутов

/ban [<Участник>] <Причина> <d|h|m|y|w>(Время: день, час, месяц, год, неделя.)

/unban [<Участник>] <Причина>

/mute [<Участник>] <Причина> <d|h|m>(Время: день, час, минута.)

/unmute [<Участник>] <Причина>

/voteunmute [<Участник>] <Причина>

/voteunban [<Участник>] <Причина>

/vote <unban|unmute> <Количество голосов>

## Администраторы (ban/unban/voteunban)

**Главный администратор**
- Количество банов: неограниченно
- Максимальная длительность: permanent (перманентный бан)
- Особые права: может отменять баны любых администраторов

**Старший администратор**
- Количество банов: 15 в день
- Максимальная длительность: 1 год 
- Может банить за серьезные нарушения

**Администратор**
- Количество банов: 10 в день
- Максимальная длительность: 1 месяц
- Стандартные модерационные права

**Младший администратор**
- Количество банов: 5 в день
- Максимальная длительность: 1 неделя 
- Ограниченные права для новых администраторов

## Модераторы (только mute/unmute/voteunmute)

**Главный модератор**
- Количество мутов: 20 в день
- Максимальная длительность: 7 дней
- Координирует работу модераторской команды

**Старший модератор**
- Количество мутов: 10 в день
- Максимальная длительность: 12 часов
- Опытный модератор с расширенными правами

**Модератор**
- Количество мутов: 5 в день
- Максимальная длительность: 6 часов
- Стандартная модерация чата

**Младший модератор**
- Количество мутов: 1 в день
- Максимальная длительность: 1 час
- Испытательный период, базовые права

## Дополнительные правила

- Все лимиты обновляются в 00:00 UTC ежедневно
- За злоупотребление правами следует понижение в должности
- Каждый бан/мут должен иметь обязательную причину
- Логи всех действий сохраняются для аудита старшими администраторами
