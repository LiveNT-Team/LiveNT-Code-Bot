# Руководство по переменным окружения

По умолчанию переменные окружения подгружаются из файла `.env` в корневом каталоге

- `BOT_TOKEN`
  Токен дискорд бота

- `AI_API_URL`
  Адрес айпи вида OpenAI API для взаимодействия с ИИ

- `AI_API_KEY`
  Ключ для ИИ айпи

- `MYSQL_USER`
  Имя пользователя MySQL

- `MYSQL_HOST`
  Хост на котором развёрнут MySQL. Если вы планируете запускать встроенный контейнер с сервером MySQL, следует поставить значение `mysql`.

- `MYSQL_PASSWORD`
  Пароль для пользователя MySQL

- `MYSQL_DATABASE`
  Имя основной базы данных

- `MYSQL_ROOT_PASSWORD`
  Пароль для root-пользователя MySQL
