# Руководство по переменным окружения

По умолчанию переменные окружения подгружаются из файла `.env` в корневом каталоге

## Бот

- `BOT_TOKEN`
  Токен дискорд бота

- `DEBUG` (`1` или `0`)
  Если значение равно `1`, **бот запустится в режиме отладки**. Если вы не планируете использовать debugpy для отладки, или запускаете бота в проде, следует поставить `0`.

## ИИ

- `AI_API_URL`
  Адрес айпи вида OpenAI API для взаимодействия с ИИ

- `AI_API_KEY`
  Ключ для ИИ айпи

## База данных

- `MYSQL_USER`
  Имя пользователя MySQL

- `MYSQL_HOST`
  Хост на котором развёрнут MySQL. Если вы планируете запускать встроенный контейнер с сервером MySQL, следует поставить значение `mysql`.

- `MYSQL_PASSWORD`
  Пароль для пользователя MySQL

- `MYSQL_DATABASE`
  Имя основной базы данных

- `MYSQL_ROOT_PASSWORD`
  Пароль для root-пользователя MySQL

## Порты

- `MYSQL_PORT`
  Порт для базы данных MySQL, рекомендуется `3306`

- `DEBUGPY_PORT`
  Порт для сервера отладки, рекомендуется `1111`
