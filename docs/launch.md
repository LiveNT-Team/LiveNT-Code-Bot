# Запуск

Убедитесь что в вашей системе установлен Docker V2

## Linux

1. Перейдите в каталог проекта
2. Сделайте копию файла `template.env` и переименуйте его в `.env`
  ```bash
  cp template.env .env
  ```

3. Укажите необходимые вам значения для переменных окружения. Для этого отредактируйте файл `.env` используя любой текстовой редактор. [Руководство по переменным окружения](./docs/env.md).
4. Соберите проект
  ```bash
  docker compose build
  ```

5. Запустите контейнеры
  ```bash
  docker compose up
  ```

> **Заметка**
>
> Для запуска в фоновом режиме добавьте флаг `-d`
