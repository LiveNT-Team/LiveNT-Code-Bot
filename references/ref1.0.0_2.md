# Техническое задание №2
Техническое задание номер 2, версия 1.0.0

## Система банов и мутов

Бот должен содержать функционал для выдачи и снятия специальной бан-роли|мут-роли, которая ограничивает действия пользователя. Эта роль должна сохранятся даже после перезахода на сервер. Бан-роли|Мут-роли будет доступен только один канал с информацией как обжаловать бан и т.п.. Если такой роли нет, бот должен предложить её создать

Могут использовать только с ролью Разработчик:

/set ban_role <роль>

/set mute_role <роль>

/set gl_admin_role <роль>
/set st_admin_role <роль>
/set admin_role <роль>
/set ml_admin_role <роль>

/set gl_moder_role <роль>
/set st_moder_role <роль>
/set moder_role <роль>
/set ml_moder_role <роль>

Могут использовать только с ролями:
Главный администратор
    └── Старший администратор
          └── Администратор
                └── Младший администратор

/ban [<Участник>] <Причина> <d|h|m|y|w>(Время: день, час, месяц, год, неделя.)

Примечание: Только главный администратор может снимать чужие баны, остальные баны могут снимать только те баны что сами наложили исключение: совместное голосование

/unban [<Участник>] <Причина>

Голосование за разбан участника при достижении числа указанного в /vote unban

/voteunban [<Участник>] <Причина>

Могут использовать только с ролями:
Главный модератор
    └── Старший модератор
          └── Модератор
                └── Младший модератор


/mute [<Участник>] <Причина> <d|h|m>(Время: день, час, минута.)

Примечание: Только главный модератор может снимать чужие муты, остальные могут снимать только те муты что сами наложили исключение: совместное голосование

/unmute [<Участник>] <Причина>


Голосование за размвт участника при достижении числа указанного в /vote unmute

/voteunmute [<Участник>] <Причина>


Только роль разработчик, лидер может изменять количество
голосов

/vote <unban|unmute> <Количество голосов>

## Система администраторов (ban/unban/voteunban)

**Главный администратор**
- Количество банов: неограниченно
- Максимальная длительность: permanent (перманентный бан)
- Особые права: может отменять баны любых администраторов
